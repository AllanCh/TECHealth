Para instalar las dependencias del proyecto: Abrir una terminnal en la carpeta
TECHealth y correr siguientes comandos:
> npm install express morgan mongoose multer
> npm install nodemon -D

Para correr el servidor en modo desarrollo:
> npm run dev
(importante tener corriendo la base datos mongo)

Rutas actuales disponibles:
> http://localhost:3000/api/offer
> http://localhost:3000/api/bookStore
> http://localhost:3000/api/book
